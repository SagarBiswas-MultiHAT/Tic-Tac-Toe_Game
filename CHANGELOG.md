# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-02-01

### Added

- GUI version using Tkinter.
- Difficulty levels (easy, mid, hard) with optimal AI on hard.
- CI checks for linting, formatting, typing, and tests.
- Packaging metadata and console scripts.

### Changed

- Refactored core game logic into modules.

### Removed

- Deprecated legacy entry-point script.

## [1.2.1] - 2026-02-01

### Added

- Series play with total matches, random first starter, and alternating starts.
- GUI scoreboard and match controls.
